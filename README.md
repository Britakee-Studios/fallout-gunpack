## Dependencies & Installation
![Dependency info](https://q1.boheurl.com/lesraisins/i/2025/01/08/fallout_dependency.png)

- Place the `.jar` file in your `mods` folder **(not** tacz gunpack path).
- Requires **TaCZ 1.1.6+** for **Minecraft 1.21.1** (NeoForge loader).

| Minecraft | Addon Version | TaCZ Version |
|-----------|---------------|--------------|
| 1.21.1    | 0.1.0         | 1.1.6+       |
